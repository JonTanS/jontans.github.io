Partition Definition

$A$ is a partition of finite set $S$:

     If $A = \{A_{1},A_{2},...A_{t}\}$ where $A_{i} \subseteq S, \forall $ $i = 1, ..., t$ such that:

       1. $A_{i} \neq \emptyset $, $ \forall $ $i = 1, ..., t$

       2. $A_{i} \cup A_{j} = \emptyset, \forall $ $i \neq j$

       3. $\bigcup\limits_{i=1}^{t} A_{i} = S$

Addition

If $A = \{A_{1},A_{2},...A_{t}\}$ is a part of $S$, then $\vert S \vert = \vert A_{1} \vert + \vert A_{2} \vert + ... + \vert A_{3} \vert$.

Multiplication

If $S = A \times B = \{(a,b)\ \vert \ a \in A, b \in B\} $ then $\vert S \vert = \vert A \vert \vert B \vert$
